RD Station Edition

Padronização exata de tipografia, cores, ícones e blocos estruturais para criação de Landing Pages de alta conversão para a marca Eternit.

Tutorial: Como usar este guia na RD Station?

Siga os 3 passos abaixo para aplicar o Design System na sua Landing Page em menos de 2 minutos.

1

Configure a Base Global

Role até a seção "Código Base Global" e copie o bloco principal. Na edição da sua Landing Page na RD Station, vá em Edição > Avançado > CSS Customizado e cole o código lá.

2

Arraste seus Textos

Com o CSS Base configurado, você só precisa usar o editor normal da RD. Arraste um bloco de texto, escolha se ele é H1, H2, H3 ou Parágrafo, e o sistema já puxará a fonte Martian B Thai e o tamanho correto sozinho.

3

Insira Ícones Facilmente

Na seção de Ícones, você tem duas opções: copiar o código HTML se souber usar componentes avançados, ou clicar para Baixar em PNG e subir na RD Station usando o bloco padrão de "Imagem".
